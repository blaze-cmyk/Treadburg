## âš™ï¸ Settings Explained

### Enable Glass UI
- **ON**: Glass effect active
- **OFF**: Normal interface

### Background Blur
- **ON**: Panels have blur effect (recommended)
- **OFF**: Panels are just transparent

### Blur Intensity
- **Low (4-8px)**: Subtle blur, better performance
- **Medium (10-16px)**: Balanced (recommended)
- **High (18-24px)**: Maximum blur effect

### Dark Overlay
- **ON**: Adds dark tint for better text readability
- **OFF**: No tint, brighter appearance

### Overlay Opacity
- **Low (0-15%)**: Light tint
- **Medium (20-30%)**: Balanced (recommended)
- **High (35-50%)**: Dark tint

### Q: Does it slow down the app?
**A:** No! Glass UI uses GPU-accelerated CSS. Very efficient.

### Q: Can I use my own background image?
